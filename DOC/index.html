Weather API Documentation

Welcome to the Weather API documentation. This guide provides all the information you need to use the API effectively.

Getting Started

To start using the Weather API, follow these steps:

  1. Install the project dependencies as described in the README file.
  2. Run the development server using the command: python manage.py runserver.
  3. Access the API at http://127.0.0.1:8000/.

API Endpoints

Below are the available endpoints:

  • GET /weather/: Fetch weather data.
  • POST /weather/: Add new weather data.

Examples

Fetching Weather Data

GET /weather/
Response:
{
    "location": "New York",
    "temperature": "15°C",
    "condition": "Cloudy"
}

Adding Weather Data

POST /weather/
Request Body:
{
    "location": "Los Angeles",
    "temperature": "25°C",
    "condition": "Sunny"
}
Response:
{
    "message": "Weather data added successfully."
}
